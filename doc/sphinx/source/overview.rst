 Overview

What is Patro ?
---------------

Patro is a Python module which implements pattern drafting for fashion design.

Which platforms are supported by Patro ?
----------------------------------------

Patro runs on Linux, Windows 64-bit and Mac OS X.

How to install Patro ?
----------------------

The procedure to install Patro is described in the :ref:`Installation Manual <installation-page>`.

Which version of Python is required ?
-------------------------------------

Patro requires Python 3 and the version 3.5 is recommended so as to benefit from the new *@* syntax
for units.
